# sound-utils


## Overview

Basic toolset to work with audio, like files and playing sound.

## Provided libraries

- `Soundfile`: Work with audio files, read/write 
- `Sounddevice`: Play audio 
- `Portaudio`: lower level audio i/o

## Dependencies

`python` and `torch` + `torchaudio`
